TrustTrack - Fraud-Proof Attendance Tracker

A fun AI-powered attendance system that makes cheating attendance hard and keeps your privacy intact. Uses live camera capture, background blur, and Firebase logging.

What it does

Captures attendance live from your camera

Detects faces so only real people get marked

Blurs background for privacy

Logs everything to Firebase

Dashboard to see attendance records

Tech Stack

Frontend: HTML, CSS, JavaScript, React

AI/ML: TensorFlow.js for face detection

Backend: Firebase (Auth + Firestore)

Hosting: Firebase Hosting

How to run

Clone the repo:

git clone https://github.com/amulyarstar/Trust-track-fraud-proof-attendance-system.git
cd trusttrack


Install dependencies:

npm install


Run locally:

npm start


Build for deployment:

npm run build


Try it out

Live MVP: https://fraud-attendance-mvp.web.app/
